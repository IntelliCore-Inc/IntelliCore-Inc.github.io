# 龙芯智造（上海）科技有限公司

## 公司简介

龙芯智造（上海）科技有限公司（IntelliCore）成立于2025年，是一家专注于智能运动硬件和运动数据分析的创新型科技公司。我们致力于通过先进的传感器技术和人工智能算法，为运动爱好者提供专业的数据化训练解决方案。

## 公司使命

用科技驱动运动，用数据赋能成长。我们相信科技可以让运动训练更加科学、高效，帮助每一位运动爱好者实现自我突破。

## 核心产品

### Swing Core 智能网球训练系统

Swing Core是我们自主研发的专业网球智能训练系统，通过高精度传感器实时采集挥拍数据，配合AI智能分析，为网球爱好者提供全方位的训练解决方案。
