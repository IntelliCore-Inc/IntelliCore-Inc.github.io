解答您关于Swing Core的疑问

产品相关

Swing Core 如何安装在网球拍上？

Swing Core 传感器模块设计为嵌入式，可直接安装在网球拍拍柄底部。安装简单，只需拧开拍柄底盖，插入传感器模块即可，不影响握感和平衡。适用于大多数标准网球拍，包括Wilson、Head、Babolat、Yonex等主流品牌。

电池续航如何？充电方便吗？

Swing Core 采用CR2032纽扣电池，待机时间长达30天，连续使用可达8小时。电池更换简单，可通过模块底部的电池仓快速更换，无需额外充电设备。

模块重量会影响打球吗？

不会。Swing Core 模块重量小于20g，安装在拍柄底部后几乎无感，不会影响球拍的平衡和您的握拍感受。我们的设计目标是真正做到"无感佩戴"。

支持哪些品牌的网球拍？

Swing Core 适配市面上绝大多数标准网球拍，包括Wilson、Head、Babolat、Yonex、Prince等主流品牌。只要拍柄底部可拆卸即可安装。

APP 相关

APP 是否支持离线使用？

是的，Swing Core 支持完全离线使用。传感器数据通过蓝牙传输至手机后暂存本地数据库，待网络恢复后自动同步至云端，确保训练记录不丢失。

如何配对传感器？

确保手机蓝牙开启，打开Victo APP的"运动"页面，点击"添加设备"按钮，在可用设备列表中选择您的Swing Core设备，等待配对成功提示即可。首次配对后，后续使用会自动连接。

数据会丢失吗？

不会。您的所有数据都会保留在本地和云端，会员到期只是限制部分高级功能的使用，您的数据仍然安全存储。续费后可继续使用全部功能。

可以在多个设备上使用吗？

可以。一个账号可以同时登录多个设备，数据会自动同步。您可以在手机上查看训练记录，在平板上进行详细分析。
